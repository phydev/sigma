# Σ-Project

Sigma is an API for validating and retrieving information from Norwegian ID numbers.


## Repository structure

The repository is structured as follows:

```
.
├── app
│   ├── __init__.py
│   ├── main.py
│   ├── functions.py
├── tests
│   ├── __init__.py
│   ├── test_main.py
│   ├── test_functions.py
├── docs
├── data
│   ├── fnr.txt
├── Dockerfile
├── docker-compose.yml
├── README.md
```
